# tinydash

TinyDash는 Lodash에서 영감을 받은 경량화되고 간소화된 JavaScript 유틸리티 라이브러리로, JavaScript 코딩을 더 쉽고 효율적으로 만들어줍니다. 배열, 객체, 문자열 작업에 일반적으로 사용되는 유틸리티 함수 세트를 제공하여, 소형에서 중형 크기의 웹 프로젝트에 완벽한 선택입니다.

## 특징

- 배열, 객체, 문자열을 다루기 위한 사용하기 쉬운 API 제공
- 외부 의존성 없는 경량화
- 누구에게나 열려있는 오픈소스

## 설치

```bash
npm install tinydash
```

## 사용법

```js
const tinydash = require("tinydash");
```
